## Sandbox

This image downloads lots of compilers used for building container used to run the online judge service.

```bash
docker pull yjl9903/sandbox
```

Support compilers:

+ gcc 12.1.0
+ g++ 12.1.0
+ clang 14.0.6
+ clang++ 14.0.6
+ openjdk 17.0.5
+ go 1.19.5
